Introduction

This web page explains some basic Unix shell commands. For these examples Bash has been used.


How to list the files in a directory

To list all of the files contained within the current directory use the command:

ls


How to show and change the working directory

To show the location of the current directory use the command:

pwd

To change directory use the command:

cd [add file path here]

To go up one level in the file structure use the command:

cd ..


How to create a new directory

To create a new directory use the command:

mkdir [add new directory name here]

Bash terminal showing use of the ls, cd and mkdir commands in the making of this web page.


How to create and remove files

To create a new text file and open the file editor use the command:

nano [add file name here eg filename.txt]

This command will open the text editor. Use the commands shown at the bottom of the text editor.


To create an empty new file use the command:

touch [add file name here eg filename.txt]

To remove a file use the command:

rm [add file name here]

The -i flag can be added to the rm command to confirm before deletion of a file.
